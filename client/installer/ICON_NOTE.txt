Place your dotachi.ico file in this directory.

Requirements:
  - File name: dotachi.ico
  - Format: Windows ICO
  - Recommended sizes (multi-resolution ICO):
      256x256
      128x128
      64x64
      48x48
      32x32
      16x16

How to create:
  1. Start with a 256x256 PNG image
  2. Convert to ICO using one of:
     - https://convertio.co/png-ico/
     - ImageMagick: magick convert icon.png -define icon:auto-resize=256,128,64,48,32,16 dotachi.ico
     - GIMP: Export as .ico with multiple sizes

The NSIS installer script (dotachi.nsi) references this icon for:
  - The installer executable icon
  - The uninstaller executable icon
